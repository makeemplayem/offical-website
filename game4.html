Bananas!

Bananas! is a fun game were you try to help a monkey get bananas by launching him into them. Instructions: point the arrow by using left/right arrow keys or a/d. adjust the speed by using r for full, t for least, up and down arrow keys for more or less speed, then press space to launch the monkey into the sky!
