block builder

block builder is the first virtual reality game ever realsesed from make'em play'em. It is a voxel based game, meaning that everything are squares. You can build structures, let you imagination run wild. It can be run with a VR headset or in fullscreen. It uses a physics engine, so blocks will fall. There is some glitches with this, so please report any you see to the help forums. You can use wasd and mouse controls to move. Have fun and let your imagination run wild.
